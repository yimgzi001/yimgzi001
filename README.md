## 我是谁

我主要写 Java 后端。

平时更关心服务怎么设计、接口怎么写、数据怎么存，以及一个项目上线之后能不能安稳地跑起来。现在也在把 AI 加进这些东西里，想看看大模型、RAG 和 Agent 到底能不能在实际项目里帮上忙，而不只是做个聊天 Demo。

目前还在积累阶段，很多东西会边学边做。这个主页和下面的仓库，主要用来记录我做过的实验和一些还没完全做好的项目。
